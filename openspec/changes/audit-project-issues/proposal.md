# Proposal: 项目结构与代码问题全局审计

## 需求

对智能外呼系统全部组件进行全局扫描，发现代码质量、架构结构、安全隐患、性能问题，输出按严重程度排序的问题清单。

## 范围

- agent-asr（ASR 服务）
- agent-tts（TTS 服务）
- agent-flow（LangGraph 编排器）
- mcp-server/java-mcp-server（MCP 服务）

## 检查维度

1. **代码质量** — 代码风格、重复代码、异常处理、潜在 bug
2. **架构结构** — 模块耦合、职责划分、目录组织
3. **安全隐患** — 敏感信息泄露、注入风险、不安全调用
4. **性能问题** — 异步处理、资源泄漏、阻塞操作

## 输出

按严重程度排序的问题清单，含文件位置和修复建议。不直接修复代码。

## 成功标准

覆盖所有四个组件，每个问题含：严重程度、文件路径、行号、问题描述、修复建议。
